# Review and Reflect

## What challenges did you face when refactoring your code to use Bootstrap?
Just understanding the documentation. The most frustrating was seeing the limitation of bootstrap.

## How did using Bootstrap utility classes and components simplify your styling process?
Probably the only thing i liked was quickly setting the parent to flex, justify-content-center and align-item-center without having to write CSS code. Although, honestly its not that much to type either way. Maybe throwing in some margins and padding quickly.

## In what scenarios might you choose not to use Bootstrap and write custom CSS instead?
Sizing stuff.  Anything where i want precision--which is probably most of the time.
